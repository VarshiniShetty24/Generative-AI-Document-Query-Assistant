🆘 QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "Module not found" error
→ Solution: Run setup.bat/setup.sh again

Problem: "OPENAI_API_KEY is required"
→ Solution: Edit .env file, add API key, restart app

Problem: Port 8501 already in use
→ Solution: streamlit run app.py --server.port=8502

Problem: Slow responses
→ Solution: Reduce CHUNK_SIZE in .env to 512

Problem: Couldn't import faiss
→ Solution: pip install faiss-cpu

→ Full troubleshooting: See INSTALLATION.md
